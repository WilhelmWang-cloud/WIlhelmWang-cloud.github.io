# Cider
Mac端的apple music可以用一坨来形容，界面UI一坨，全屏卡顿bug一直不修复，经常黑屏。因此我推荐一款第三方接入Apple music的软件——Cider
## 安装方法

在终端输入如下指令：

`brew install --cask cider`

注：需要先安装Homebrew，具体看下一条

# 软件包管理工具Homebrew

Homebrew是macOS（或 Linux）的软件包管理器。通过它，我们可以方便的对Mac上的各种应用软件进行管理，例如：安装、更新、查看、搜索、卸载等。Homebrew可谓是Mac神器，几乎每一位Mac使用者都会安装它。

用Homebrew官网的一句话总结：

Homebrew 使 macOS（或您的 Linux 系统）更完整。

## 安装方法

在终端输入如下指令

`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

注：需要挂梯子
